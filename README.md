**前台界面：
①首页、公告管理、查看更多等
②心理健康学习、文章标题搜索、试卷列表、考试等
③公告通知、留言反馈等
④个人中心、考试记录、错题本等

①学生登录：
个人中心、修改密码、个人信息、辅导预约管理、考试管理（考试记录、错题本、试卷列表）
②教师登录：

个人中心、修改密码、个人信息、学生管理、教师管理、辅导预约管理、学生预约管理、测试结果分析管理、心理健康学习、试题管理、试卷管理、系统管理、留言板管理、考试管理（考试记录）**
